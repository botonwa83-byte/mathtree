# Changelog

## [Unreleased]

### Added

- **公式证明增强**：为核心公式添加了详细证明步骤，包括：
  - 勾股定理：4步面积法证明
  - 和角公式：4步复数乘法证明  
  - 余弦定理：5步辅助线构造证明
  - 欧拉公式：5步泰勒级数证明
- **交互式证明体验**：
  - 折叠/展开功能：点击步骤可展开/折叠详细内容
  - 步骤导航：上一步/下一步按钮，支持顺序浏览
  - 进度指示器：显示当前步骤位置
  - 证明摘要卡片：一键查看所有关键洞见
- **动画效果**：
  - 步骤切换的弹簧动画
  - 展开/折叠的淡入淡出效果
  - 当前步骤的缩放高亮

### Improved

- **FormulaView优化**：
  - 更新Katex版本到0.16.10
  - 添加iOS 16.4+ JavaScript权限配置
  - 优化HTML加载方式，使用动态脚本加载
  - 添加字符串转义增强

### Fixed

- 修复中文标点符号导致的编译错误
- 修复SwiftUI `.border()` 修饰符语法错误
- 修复WKWebView在iOS 17+上的运行时警告

### Technical Notes

- 每个证明步骤包含：标题、内容、LaTeX公式、视觉提示、关键洞见
- 向后兼容设计：支持旧的`proofSteps`字符串数组和新的`detailedProof`详细证明
- 借鉴了GitHub上的优秀数学可视化项目（treehehe、prooffold、proof-sketcher）
